# Passage — passagecare.co

Landing page for Passage Coordination LLC.

## Deploy

This is a static site. Push to GitHub and connect to Vercel — it deploys automatically.

## Structure

- `index.html` — the full landing page
- `vercel.json` — tells Vercel to serve as static

## Custom Domain

In Vercel dashboard → Settings → Domains → Add `passagecare.co`
